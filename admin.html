Обновление сайта

После редактирования данных скачайте обновленный файл и загрузите его на GitHub

Инструкция по обновлению:

  1. Нажмите кнопку "Скачать обновленный data.js" ниже
  2. Перейдите в репозиторий на GitHub
  3. Найдите файл data.js и нажмите на него
  4. Нажмите иконку карандаша (Edit this file)
  5. Удалите весь старый код и вставьте содержимое скачанного файла
  6. Прокрутите вниз и нажмите "Commit changes"
  7. Подождите 1-2 минуты пока GitHub Pages обновится
  8. Обновите сайт чтобы увидеть изменения
