📖 Инструкция по обновлению данных на GitHub Pages

  1. Скачайте обновленный файл: Нажмите кнопку "Сохранить изменения" чтобы скачать новый файл data.json
  2. Зайдите в репозиторий на GitHub: Перейдите в ваш репозиторий с сайтом
  3. Найдите файл data.json: В корневой папке репозитория найдите текущий файл data.json
  4. Загрузите новый файл:
    • Нажмите на файл data.json
    • Нажмите иконку карандаша (Edit)
    • Удалите весь старый текст
    • Откройте скачанный файл в текстовом редакторе, скопируйте всё содержимое
    • Вставьте новое содержимое в редактор на GitHub
    • Прокрутите вниз, напишите комментарий коммита (например: "Update site data")
    • Нажмите "Commit changes"
  5. Или через загрузку файла:
    • В репозитории нажмите "Add file" → "Upload files"
    • Перетащите скачанный data.json или выберите его
    • Нажмите "Commit changes"
  6. Подождите обновления: GitHub Pages автоматически обновит сайт в течение 1-10 минут

Важно: После загрузки обновите страницу сайта (Ctrl+F5) чтобы увидеть изменения!
